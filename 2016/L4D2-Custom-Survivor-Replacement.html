求生之路2人物模型替换[译]

Posted on 2016-04-10   |  

原帖地址: http://steamcommunity.com/sharedfiles/filedetails/?id=142457190

2016/4/11更新

开始准备

好的，让我们从最基础的开始。首先你需要安装 Steam 和 Left 4 Dead 2. 额外的我们还需要安装 Left 4 Dead 2 Authoring Tools 和 Source SDK. 你点击Seam图标，选择 库，然后从这 库 的菜单中点击 工具（如果你默认没有显示它们的话这会显示所有可用的工具）。

现在点击你所要的工具（Left 4 Dead 2 Authoring Tools 和 Source SDK）然后选择安装游戏。接着在弹窗里确认安装。在你做这些的时候我准备去玩几局昆特牌:) 因为昆特牌实在是太xx爽了。你们弄好了我就会回来。

OK，节目继续，基础的准备都准备好了，让我们将每个程序运行一次让它们初始化。如果你没有在安装时创建快捷方式那你能在Steam的库里运行它们。我们先运行 Authoring Tools… 如果没有意外的话它启动后基础的界面应该是酱紫的:

注意你现在也将 Face Poster 和 Model Viewer 安装上了。那些现在对你都不重要。
现在让我们关闭这个程序然后启动 Source SDK. 第一次启动你会看见一个”copying files please wait…”的窗口，耐心等待它结束。当它运行完毕后会打开一个浏览窗口还有一些我压根不关心的公告，直接右上角点X让它滚蛋…现在的界面和 Authoring Tools 非常的相似:

请注意 Source SDK 里的 Model Viewer 和 Face Poster 与 Authoring Tools 里的 不完全相同。这个东西你暂时还用不着关心。现在只需要记住就好了。 OK, 现在我们所需要的东西都安装运行的不错了，那我们继续。

模型处理

OK，如果我们想对模型进行修改我们需要能够打开这些模组和各种各种样的模组文件格式。还需要将它们导出和导入我们的游戏。那我们就先把埃利斯从求生之路2里提取出来开始。

首先我们需要 GCFScape 这个程序。
这里是下载链接:http://nemesis.thewavelength.net/files/files/gcfscape183.exe

这是个可执行文件，但我保证它不是病毒。如果你不相信你可以自己百度这个链接来满足你的妄想症(原文为If you don’t trust me you can go Google the bloody link yourself Captain Paranoia.)。运行安装程序并让它自己选择默认安装选项(这样是为了让我们的安装的环境都一样，别到时候哭着来说自己和我的不一样做不了，因为你非要买萌地把它安装在 “你个坑货” 文件夹里)。

原文为:so don’t come crying to me when something doesn’t work because you
wanted to be cute and install it in the “FudgePuddles” folder on your
computer.

一旦它安装好了你就在 开始菜单->所有程序->Nem`s Tools->GCFScape 运行它。打开后大概就像这样:

OK, 然后让我们把那个乡巴佬(redneck 乡下人 这里指埃利斯)抓出来。点击 文件(File)->打开(Open) 或者点击左上角的文件夹图标。我们需要打开保存求生之路2里人物模型VPK文件的文件夹(你可以把VPK想象成一个ZIP压缩文件)。

这些文件的路径就像这样:
C:\Program Files\Steam\steamapps\common\left4dead2\left4dead2\pak01_dir.vpk

注:每个人的路径可能不一样，你需要到安装求生之路2时的路径里寻找

找到的文件如图所示:

选择这个文件(pak01_dir.vpk)然后点击打开(Open)，你能得到这东西:

这些就是这个VPK文件包含的所有东西了。你能在里面找到所有的地图模型、纹理脚本和别的好东西。我们将在这里把 埃利斯提取出来。这个文件相当重要，对以后你自己做VPK插件展示了正确的文件结构，返回去再看一次这个文件结构吧。

我们继续…在左边的窗口里，点击 models 旁边的加号( + )，往下滑再点击 survivors 旁边的加号。在右边的窗口里你能看见许多文件。如果你在右边窗口里寻找你能看见一些叫做”survivor_machanic”的文件，那个就是我们亲爱的埃利斯。接着我们酱紫做：

  1. 让我们在右边的窗口里将这些文件标出来(你能得到五个这样的文件)。
  2. 右键选中它们然后选择提取(Extract)，它会询问你解压到哪。让我们养成一个好习惯把它们都放在一个合适的位置…那点击桌面。
  3. 新建文件夹，把它命名为”survivor_ellis”然后标记它。
  4. 再次新建文件夹，命名为”models”。
  5. 再次新建文件夹，命名为”survivors”
  6. 选中这个文件夹点击OK。

我们刚才创建了一个和VPK文件相似结构的文件夹，因为在VPK文件里这些文件都是在 models\survivors\ 这个文件夹里。有感觉了么？我实在是词穷不知道怎么解释这个了，直接看提取后的路径图吧:

现在，庆祝一下吧，让我们把埃利斯的表面材质纹理文件提取出来(let’s grab Ellis’s main texture files. )。

  • 返回GCFScape程序，这次我们点击materials旁边的加号。
  • 然后点击models旁边的加号。
  • 再点击survivors旁边的加号。
  • 在右边的窗口你能看见一些文件夹，注意器中一个叫做 mechanic, 右键它选择提取。
  • 这次还是会询问你提取到哪个地方，同样的我们照着VPK文件的结构放置它。
  • 于是我们将它保存在survivor_ellis\materials\models\survivors(将mechanic文件夹放在survivors文件夹里，这样mechanic文件夹里所有的文件都在里面)。
  • 这样我们就得到了我们所需要埃利斯所有的模型和纹理文件。
  • 关闭 GCFScape程序。
