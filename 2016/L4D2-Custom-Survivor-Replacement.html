Posted on 2016-04-10   |  

原帖地址: http://steamcommunity.com/sharedfiles/filedetails/?id=142457190

开始准备

好的，让我们从最基础的开始。首先你需要安装 Steam 和 Left 4 Dead 2. 额外的我们还需要安装 Left 4 Dead 2 Authoring Tools 和 Source SDK. 你点击Seam图标，选择 库，然后从这 库 的菜单中点击 工具（如果你默认没有显示它们的话这会显示所有可用的工具）。

现在点击你所要的工具（Left 4 Dead 2 Authoring Tools and Source SDK）然后选择安装游戏。接着在弹窗里确认安装。在你做这些的时候我准备去玩几局昆特牌:) 因为昆特牌实在是太xx爽了。你们弄好了我就会回来。

OK，节目继续，基础的准备都准备好了，让我们将每个程序运行一次让它们初始化。如果你没有在安装时创建快捷方式那你能在Steam的库里运行它们。我们先运行 Authoring Tools… 如果没有意外的话它启动后基础的界面应该是酱紫的:

注意你现在也将 Face Poster 和 Model Viewer 安装上了。那些现在对你都不重要。
现在让我们关闭这个程序然后启动 Source SDK. 第一次启动你会看见一个”copying files please wait…”的窗口，耐心等待它结束。当它运行完毕后会打开一个浏览窗口还有一些我压根不关心的公告，直接右上角点X让它滚蛋…现在的界面和 Authoring Tools 非常的相似:

请注意 Source SDK 里的 Model Viewer 和 Face Poster 与 Authoring Tools 里的 不完全相同。这个东西你暂时还用不着关心。现在只需要记住就好了。 OK, 现在我们所需要的东西都安装运行的不错了，那我们继续。
